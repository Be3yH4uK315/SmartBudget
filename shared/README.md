# Общие библиотеки и модули (shared)

Папка содержит общие зависимости, которые используются несколькими микросервисами.  
Это позволяет избежать дублирования кода и унифицировать бизнес-логику, DTO и схемы данных.

## 📁 Структура папки
```
shared/
├── python/
│ ├── models/ # Общие Pydantic / SQLAlchemy модели
│ ├── utils/ # Утилиты (хэширование, логирование, JWT)
│ ├── clients/ # Общие HTTP/gRPC клиенты
│ ├── schemas/ # DTO и сериализация
│ └── setup.py # pip-пакет (например, shared-utils)
├── dotnet/
│ ├── Models/ # Общие DTO и модели для C#
│ ├── Utils/ # Хелперы и расширения
│ ├── Clients/ # Общие HTTP/gRPC клиенты
│ └── Shared.csproj
└── README.md
```
